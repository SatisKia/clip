# clip.js

CLIPインタープリタ用の計算処理エンジンです。

## ビルド方法

build.batを実行しますと、coreフォルダ下にclip.jsとclip.debug.jsが生成されます。

ビルドには別途、次のツールが必要です。

### MinGW

UTF-8対応Cプリプロセッサとして使用します。

### Microsoft Ajax Minifier

JavaScriptコードの圧縮・難読化ツールです。

## テスト

### math

数値計算テストプログラムです。

### core

CLIPインタープリタです。
