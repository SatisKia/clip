2021年1月24日
言語仕様関連
	多倍長演算を追加
	+----------+-----------------------------------------------------------+
	| 変数表現 | @@char  @@:code  @@:0または@で最後の演算結果が得られる    |
	+----------+-----------------------------------------------------------+
	| コマンド | :mfloat  :mint                                            |
	+----------+-----------------------------------------------------------+
	| 演算子   | ++  --  !  [++]  [--]  [-]  [+]  **                       |
	|          | *  /  %  +  -  <  <=  >  >=  ==  !=  ?                    |
	|          | =  *=  /=  %=  +=  -=  **=  ,                             |
	+----------+-----------------------------------------------------------+
	| 内部関数 | pow  sqr  sqrt  int  abs  fact                            |
	+----------+-----------------------------------------------------------+
	| ECLIPSE  | $INC  $DEC  $NEG  $MINUS  $SET  $SET_Z                    |
	|          | $MUL  $MUL_A  $DIV  $DIV_A  $MOD  $MOD_A                  |
	|          | $ADD  $ADD_A  $SUB  $SUB_A  $POW  $POW_A                  |
	|          | $LT  $LT_A  $LE  $LE_A  $GT  $GT_A  $GE  $GE_A            |
	|          | $EQ  $EQ_A  $NEQ  $NEQ_A  $CND                            |
	+----------+-----------------------------------------------------------+
	演算子"**","**="の右辺は整数値
	演算子"%","%="の左辺、右辺は整数値
	内部関数"pow"の第二パラメータは整数値
	対応するECLIPSEにも同様の制限があります。
